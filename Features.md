# Features
## Module: Admin - Program Check-in for Opted-in Users
| ID | Feature | Description |
|---|---|---|
| 1 | Admin - View List of Donations | Admin should be able to view a list of donations along with details of the donor, including information such as donor name, material donated, and donation details. |
| 2 | Admin - Assign Donation(s) to Program (Single or Bulk Update) | Admin should have the option to assign one or multiple donations to a program. This feature should support both single and bulk update functionalities. |
| 3 | Admin - Search by Material Name and Bulk Assign to Program | Admin should be able to search for donations by material name and bulk assign them to a program. This feature streamlines the process of assigning multiple donations with similar materials to a program. |
| 4 | Admin - Search by Donor Name and Assign Donation(s) to Program |Admin should have the ability to search for donations by donor name and assign their donation(s) to a program. This feature facilitates the assignment of donations from specific donors to relevant programs. |
| 5 | Admin - Filter by Program and Upload Group Image | Admin should be able to filter donations by program and upload a group image specific to that program. This feature enhances program management by allowing admins to associate images with each program for visual representation. |
| 6 | Admin - Upload Image for Donor or Donation | Admin should have the option to upload images specific to individual donors or donations. This feature enables admins to provide visual documentation or representation for donors or specific donations. |
| 7 | Admin - Search for Donor or Donation and Upload Image  | Admin should be able to search for a donor or a donation and upload an image specific to that donor or donation. This feature facilitates the association of images with specific donors or donations for better organization and documentation. |
