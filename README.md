# Material Donation Registration Form

## Overview
Translate the prototype Material Donation Registration Form design from Figma into a functional and interactive React component
This project implements a Material Donation Registration Form in ReactJS for BWorks . The form captures information about material donations, focusing on bicycles and computers.

## Features

- **Objective:** Create a user-friendly form based on Figma design.
- **Components:** React component for form rendering.
- **Styling:** Responsive design using Tailwind CSS.
- **Functionality:** Form validation and submission logic.
- **Integration:** BWorks image seamlessly included in the form.
- **Testing:** Thorough testing for functionality and styling.
- **Documentation:** Clear documentation for code understanding.
- **Review:** Feedback and improvements from teammates.

## Setup

1. Clone the repo.
2. Install dependencies with `npm install`.
3. Run the dev server using `npm start`.

## Usage

Access the form through the specified route. Users can fill in details and submit.
