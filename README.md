# material-donor-mutual-assist
This is the repository for Material Donor Mutual Assist project.


## Instructions

### To run client app, admin-module, and donor module

- cd to respective folder
  - <code> cd backend/admin-module</code> or
  - <code> cd backend/donor-module</code> or
  - <code> cd client-app</code>
- run <code>npm install</code>
- run <code>npm start</code>

### To run test cases

(These can be run with or without the above steps)
- To run all tests  
  <code>npm test</code>
- To run mock tests for admin-module/crud_donors.js 
  <code>npm test:crud-donors-mock</code>
