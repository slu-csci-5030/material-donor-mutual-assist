# BWorks Web Application

This is a web application developed for [BWorks company](https://www.bworks.org/). The purpose of this application is to improve student skills and facilitate the donation process for both adult and children’s sized bicycles and computers. BWorks operates various programs to allocate these donations to children in need.

## Getting Started

To run the project locally, follow these steps:

1. **Frontend Setup**:
   - Navigate to the `client-app` directory.
   - Install dependencies using npm:
     ```bash
     npm install
     ```
   - Start the frontend server:
     ```bash
     npm start
     ```

2. **Backend Setup**:
   - Navigate to the `backend/donor-module` directory.
   - Ensure Docker is installed on your system.
   - Run the following command to set up the backend and access the dataset:
     ```bash
     docker-compose up
     ```

## Repository Structure

- `client-app`: Contains the frontend codebase.
- `backend/donor-module`: Contains the backend codebase.
