# Donor Tracking System

This project is a simple Donor Tracking System application built with React.

## Running the Application

Follow these steps to run the application:

1. *Install Dependencies*: Run npm install inside client-app directory to install all necessary dependencies.

2. *Start the Development Server*: Run npm start to start the development server. This will launch the application in your default web browser.

## Requirement

The testing requirement for this project is outlined in [GitHub Issue #43](https://github.com/slu-csci-5030/material-donor-mutual-assist/issues/43)

## Testing the Application

To test the application, follow these steps:

1. *Run Tests*: Run npm test inside client-app directory to execute the test suite. This will run all test cases and provide feedback on their success or failure.

2. *View Test Results*: After running the tests, the test runner will display the results in your terminal. Failed tests will provide information about what went wrong.
